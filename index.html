What would you like to do?

Produktdaten aus Amazon übersichtlich darstellen

Ausgangslage

Für das Modul Webprogrammierung hat die Projektgruppe den Auftrag erhalten, eine Applikation zur Datenvisualisierung zu bauen. Die Daten werden durch eine API angezogen und anschließend in JavaScript verwendet.

Datennutzung

Für das vorliegende Projekt hat sich die Gruppe entschieden, mittels der Rainforest API Daten, welche von Amazon gescrapet wurden, zu visualisieren. Dabei wird der Hauptfokus in der Anzeige der Produkte liegen. Zudem kann sich die Projektgruppe noch vorstellen, folgende Daten einzubauen: - Kundenreviews - Suchergebnisse

Visualisierung

Die Projektgruppe kann sich vorstellen, die Daten aus der API folgendermaßen darzustellen: - Liniendiagramm, wie sich das Kaufverhalten über die Jahre verändert hat (vgl. Bloomberg) - Weltkarte, welche Produkte in welchen Teilen der Welt am meisten gekauft werden

Verwendungszweck

Im Semester 7 wird sich die Projektgruppe mit dem Erstellen eines eCommerce Stores in Shopify auseinandersetzen. Diese Webapplikation soll eine Grundlage schaffen, um die Produktrecherche so einfach wie möglich zu gestalten.
